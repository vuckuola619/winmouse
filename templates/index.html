Instructions & Tips

  • Design Mode: Choose "Pixel Canvas" to draw pixel art, or "AI Generate" to create a cursor from text prompts.
  • Hotspot: The hotspot is the active click pixel. Left-click anywhere on the workspace canvas to position the red crosshair.
  • Apply to Windows: Click "Export to Cursor" to immediately set the cursor for the active cursor type.
  • Administrator Warnings: If cursors fail to apply, ensure Flask is running with permission to modify registry path HKEY_CURRENT_USER\Control Panel\Cursors.
